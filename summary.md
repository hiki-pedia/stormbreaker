풍력발전은 풍속, 풍향, 기온 등 기상 조건에 따라 발전량이 크게 달라지기 때문에, 실제 운영 환경에서는 기상예보 데이터를 바탕으로 향후 발전량을 정밀하게 예측하는 역량이 필요합니다.

특히 발전량 예측 정확도는 전력계통 운영의 안정성뿐만 아니라, 재생에너지 발전량 예측제도에 따른 정산금 확보와도 직접적으로 연결됩니다.

'제3회 풍력발전량 예측 AI 경진대회 - BARAM 2026'은 실제 풍력발전 데이터를 기반으로, 참가자들이 기상예보 정보를 활용해 발전량을 예측하는 AI 모델을 개발하고, 재생에너지 분야에서 활용 가능한 데이터 분석 역량을 겨루기 위해 마련되었습니다.

[리더보드 평가 및 심사 결과]
1. 리더보드
평가 산식 : 총점(Score) = 0.5 x 평균 예측오차율(1-NMAE) + 0.5 x 정산금획득률(FICR) [코드 바로가기]
평가는 실제 발전량이 설비용량의 10% 이상인 시간대만 대상으로 합니다.



1) 1-NMAE
			그룹별 NMAE = 평균( |예측 발전량 - 실제 발전량| / 그룹 설비용량 )

			각 그룹별로 예측 발전량과 실제 발전량의 차이를 계산한 뒤, 이를 그룹 설비용량으로 나누어 평균합니다.

			1-NMAE = 1 - 3개 그룹 NMAE 평균

값이 높을수록 예측 오차가 작다는 의미입니다.



2) 정산금획득률
			그룹별 FICR = 획득 정산금 / 이론상 최대 정산금

			각 시간대별 NMAE 기준에 따라 획득 정산금을 계산하고, 이를 이론상 최대 정산금과 비교합니다.

			FICR = 3개 그룹 FICR 평균

			시간별 정산금은 아래 기준에 따라 산정합니다.





Public Score : 전체 평가 데이터 중 사전 샘플링된 40%
Private Score : 전체 평가 데이터 중 나머지 60%

	- 외부 데이터는 제출된 코드와 자료만으로 재현 가능해야 합니다.

	- 외부 데이터의 생성·공개·확정 시점이 각 예측값의 예측기준시점 이전임을 설명할 수 있어야 합니다.

	- 외부 데이터의 출처, 수집·전처리 과정, 활용 가능 시점 또는 재현 가능성이 충분히 소명되지 않는 경우 평가 대상에서 제외될 수 있습니다.



5. 사전 학습모델 사용 규칙
- 2026년 7월 6일 전, 즉 2026년 7월 5일까지 공식적으로 가중치가 공개된 오픈소스 모델만 사용할 수 있습니다.

오픈소스 라이선스 모델이란 모델 가중치의 사용, 수정, 배포, 재배포 및 상업적 이용이 허용되는 라이선스로 공개된 모델을 의미합니다.
단순히 가중치가 공개되어 있더라도, 비상업적 이용·연구 목적·평가 목적 등으로 사용 범위가 제한된 모델은 사용할 수 없습니다.
- 해당 조건을 충족하지 않는 모델은 사용할 수 없습니다.

- 사용한 모델, 가중치, 라이선스, 다운로드 경로 등은 코드 검증 및 2차 평가 과정에서 확인될 수 있습니다.



6. API 기반 모델 사용 제한
- OpenAI API, Gemini API, Hugging Face Inference API, Together AI, OpenRouter 등 원격 서버를 통해 모델 응답을 받는 API 기반 모델 사용은 불가능합니다.

7. 유의 사항
- 1일 최대 제출 횟수: 5회

- 모든 CSV 형식의 데이터와 제출 파일은 UTF-8 인코딩을 적용합니다.

- 평가 데이터셋은 제출 파일(예측 결과) 생성을 위한 추론 목적으로만 사용할 수 있습니다.

- 최종 순위는 선택된 파일 중에서 채점되므로 참가자는 제출 창에서 자신이 최종적으로 채점받고 싶은 파일 1개를 선택해야 합니다.

- 대회 종료 후 공개되는 Private 랭킹은 최종 순위가 아니며, 코드 검증 및 2차 평가 후 수상자가 결정됩니다.

- 본 대회는 참가팀의 부정 행위를 금지하고 있습니다. 부정 행위가 발견되는 경우, 해당 팀에 대해 평가가 진행되지 않습니다.

- 대회 기간 중 규칙 위반 의심 정황이 발견되는 경우 주최측에서 해당 팀에게 소명 및 코드를 요청할 수 있습니다.
