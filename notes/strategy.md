# 전략

## 목표

2025년 평가 기간의 `kpx_group_1`, `kpx_group_2`, `kpx_group_3` 시간별 풍력 발전량을 예측한다.

## 핵심 원칙

- 원본 데이터 파일은 수정하지 않는다.
- 예측 시점에 사용할 수 있는 정보만 사용한다.
- 무작위 분할이 아니라 시간 기준 검증을 사용한다.
- 필요하면 세 KPX 그룹을 따로 학습하고 평가한다.
- SCADA는 과거 보조 데이터로 다루며, 2025년 추론 입력으로 직접 사용하지 않는다.
- 예측값은 유효한 설비용량 범위로 제한한다.

## 초기 우선순위

1. LDAPS/GFS 기상예보만으로 누수 없는 기준 모델을 만든다.
2. 신뢰할 수 있는 내부 검증 구조를 만든다.
3. 풍속, 풍향, 격자 집계, 달력 피처를 추가한다.
4. 그룹별 모델을 비교한다.
5. NMAE/FiCR 특성에 맞춘 후처리를 추가한다.

## 중요 데이터 제약

- `kpx_group_3` 정답은 2022년에 제공되지 않는다.
- UNISON SCADA는 2023년부터 제공된다.
- 평가 기간 SCADA와 실제 발전량은 제공되지 않으며 사용하면 안 된다.
